# 🛋️ Стол Плюс — Интернет-магазин мебели

Веб-сайт ва унсурҳои интерфейси мутобиқшаванда (Responsive UI) барои мағозаи мебелбарории «Стол Плюс», ки бо истифода аз HTML5 ва Tailwind CSS сохта шудааст. Лоиҳа комилан барои компютерҳо (Desktop) ва телефонҳои мобилӣ (Mobile) мувофиқ карда шудааст.

## 🛠️ Технологияҳои истифодашуда (Technologies)

* HTML5 — Барои сохтор ва скелети сайт.
* Tailwind CSS — Барои стилҳо, рангҳо ва таъмини мутобиқшавӣ (responsive utility classes).
* Font Awesome v6 — Барои нишонаҳо (иконкаҳо), ба мисли ситораҳо, тирчаҳо ва сабад.
* Google Fonts (Sans-serif) — Барои шрифтҳои хонданибоб ва муосир.

## 🚀 Хусусиятҳои Техникии Лоиҳа (Technical Features)

1. Mobile-First & Desktop Responsive:** Сайт аз андозаи калони компютерӣ (lg:block) то хурдтарин экранҳои телефонӣ (block lg:hidden) худкор тағйир меёбад.
2. Custom Scrollbars (Пинҳон кардани скрoлл): Дар телефонҳо ҳамаи бахшҳои уфуқӣ (скрoллшаванда) дорои стили махсуси .no-scrollbar ҳастанд, то хати хокистарии скрoлл намоён нашавад ва тарҳ зебо монад.
3. **Max Width Limit (max-w-7xl): Барои он ки сайт дар мониторҳои хеле калон (UltraWide) аз ҳад зиёд паҳну вайрон нашавад, паҳноии марказии он маҳдуд ва баробар карда шудааст.
4. Hover Interactions:** Рангҳои тугмаҳо ва линкҳо ҳангоми бурдани муш (hover) ба таври мулоим тағйир меёбанд.
